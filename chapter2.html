Number System and Conversion & Boolean Logic

The number system is a way of representing numbers in various forms, such as:

  • Binary: Base-2 number system using 0s and 1s.
  • Decimal: Base-10 number system using digits 0-9.
  • Octal: Base-8 number system using digits 0-7.
  • Hexadecimal: Base-16 number system using digits 0-9 and letters A-F.

Understanding these systems is crucial for computer science, as computers operate using binary (0s and 1s). Conversion between these systems is a fundamental skill.

Number System Conversion

Here are some common number system conversions:

  • Binary to Decimal: Convert each binary digit (bit) into its decimal equivalent, then add up the values.
  • Decimal to Binary: Repeatedly divide the decimal number by 2 and keep track of the remainders until you reach 0.
  • Hexadecimal to Binary: Each hexadecimal digit corresponds to a 4-bit binary value.
  • Binary to Hexadecimal: Group the binary digits into sets of four, starting from the right, and convert each group to its hexadecimal equivalent.

For example, to convert the decimal number 10 to binary, we divide 10 by 2:

            10 ÷ 2 = 5 remainder 0
            5 ÷ 2 = 2 remainder 1
            2 ÷ 2 = 1 remainder 0
            1 ÷ 2 = 0 remainder 1

The binary representation of 10 is 1010.

One's Complement

One's complement is a way of representing negative numbers in binary. To find the one's complement of a binary number, flip all the bits (change all 1s to 0s and all 0s to 1s).

For example, let's find the one's complement of the binary number 1010:

            Original:    1010
            One's Complement: 0101

Two's Complement

Two's complement is another way of representing negative numbers in binary. It is similar to one's complement but involves an additional step. After finding the one's complement, you add 1 to the result.

For example, let's find the two's complement of the binary number 1010:

            Original:    1010
            One's Complement: 0101
            Add 1:      0110

The two's complement of 1010 is 0110.

Boolean Logic

Boolean logic involves the use of true/false values and is essential for programming and circuit design. Key points include:

  • Forms the basis of decision-making in algorithms and programming.
  • Utilizes logical operators such as AND, OR, and NOT.
  • Helps in constructing logical expressions and conditions.

Some common Boolean expressions:

  • AND: Returns true if both conditions are true.
  • OR: Returns true if at least one condition is true.
  • NOT: Inverts the boolean value (true becomes false, and false becomes true).

1. What is the binary representation of the decimal number 15?

The binary representation of 15 is 1111. To convert, repeatedly divide by 2 and note the remainders:

                    15 ÷ 2 = 7 remainder 1
                    7 ÷ 2 = 3 remainder 1
                    3 ÷ 2 = 1 remainder 1
                    1 ÷ 2 = 0 remainder 1

The binary number is 1111.

2. How do you convert a binary number to hexadecimal?

To convert a binary number to hexadecimal:

  • Group the binary digits into sets of four (starting from the right).
  • Convert each group of four bits into a single hexadecimal digit.

Example: Binary 101101 becomes 0010 1101, which is 2D in hexadecimal.

3. What is the result of the Boolean expression: true AND false?

The result of true AND false is false, because for an AND operation, both conditions must be true for the result to be true.

4. What is the one's complement of the binary number 1010?

The one's complement of 1010 is 0101. To find the one's complement, flip all the bits.

5. How do you calculate the two's complement of the binary number 1010?

First, find the one's complement of the binary number:

                    One's Complement of 1010: 0101

Then, add 1 to the result:

                    Add 1: 0110

The two's complement of 1010 is 0110.
